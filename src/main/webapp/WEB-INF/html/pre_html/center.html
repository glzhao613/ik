公司简介

爱课培训中心创建于2018年10月10日，是一家专业从事艺术性培养和大脑潜能开发的民办教育机构和教育实体.该机构强调“全员参与的，全过程的、面向全体各个爱学习的小伙伴”。并以实现“在快乐的海洋里寻求知识的奥秘，帮助学子们快乐学习”为目标。这也是我们全体成员的美好向往。

爱课是一家新兴的培训机构，拥有前沿的培训教育方法，最优秀的培训团队。创办人是我们从美国哈佛留学回来的克里斯汀.火。他在美国曾获得过诺贝尔文学奖，同时也是天使投资人。我们的教学理念是：全程超越传统教学模式。这将是教育继续改革的延续和结晶。

创始人

爱课培训中心由克里斯汀.火创建，是麻省理工大学的高材生，于2008年毕业，毕业后一直从事于教育事业，至今拥有近十年的教育经验。曾获得诺贝尔文学奖，同年他的文章也在美国最大的报社得以发表。

创建史

爱课培训中心，简称“AK”，创始于2018年10月10日，位于贵州省贵阳市花溪区贵州大学，是一家专业从事艺术性培养和大脑潜能开发的民办教育机构和教育实体。
